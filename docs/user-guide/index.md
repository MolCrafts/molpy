# User Guide

Welcome to the MolPy User Guide! This section provides comprehensive documentation on how to use MolPy's modules.

## Modules

All modules in MolPy are built around the `core` module, which provides the fundamental data structures (`Frame`, `Block`, `Box`, `Atomistic`). Each module extends MolPy's functionality:

- **[Adapter](adapter.ipynb)**: Bridge to external libraries (RDKit)
- **[Builder](builder.ipynb)**: Molecular construction and assembly
- **[Compute](compute.ipynb)**: Unified computation abstraction
- **[Engine](engine.ipynb)**: Interfaces to simulation engines (LAMMPS, CP2K)
- **[IO](io.ipynb)**: File format readers and writers
- **[Op](op.ipynb)**: Geometric operations
- **[Pack](pack.ipynb)**: Molecular packing tools
- **[Parser](parser.ipynb)**: SMILES/SMARTS parsing
- **[Potential](potential.ipynb)**: Force field potential functions
- **[Reacter](reacter.ipynb)**: Chemical reaction modeling
- **[Typifier](typifier.ipynb)**: Automatic atom typing
