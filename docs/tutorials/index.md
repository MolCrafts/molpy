Start here if you're new to MolPy:

- **[Box Tutorial](box.ipynb)**: Learn how to create and work with simulation boxes
- **[Frame & Block Tutorial](frame-block.ipynb)**: Understand the core data structures
- **[Wrappers Tutorial](wrappers.ipynb)**: Learn about the wrapper system for extending functionality
- **[Crystal Builder Tutorial](crystal-builder.ipynb)**: Learn how to build crystal structures
